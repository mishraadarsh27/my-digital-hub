My Digital Hub
A secure, cloud-based web application to manage all your personal social media links and sensitive credentials in one place. Built with modern web technologies and backed by Google Firebase for authentication and data storage.

Live Demo: https://my-digital-hub-30516.web.app

Key Features
Secure User Accounts: Full sign-up, sign-in, and password reset functionality using Firebase Authentication.

Google Sign-In: Option for one-click sign-in with a Google account.

Encrypted Vault: A secure vault for storing usernames and passwords, protected by a personal 4-digit PIN. All data is encrypted before being saved.

Multi-Account Support: Save multiple credentials for the same social media platform.

User-Friendly Interface: A modern, attractive, and responsive design that works on any device.

Profile Customization: Users can update their name, bio, profile picture, and manage a list of public social media links.

Convenience Features:

One-click copy for usernames and passwords.

Quick links to official social media login pages.

Password visibility toggle (show/hide).

Technologies Used
Frontend: HTML5, Tailwind CSS, JavaScript (ES6 Modules)

Backend: Google Firebase

Firebase Authentication: For managing user accounts.

Firestore Database: For storing user profile data and encrypted credentials securely.

Encryption: CryptoJS for AES encryption of vault data.

Hosting: Deployed live using Firebase Hosting.
